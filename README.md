# ironhelper-kb

Knowledge-base bundle for the (private) Ironman Progression Helper RuneLite plugin.
Data only — derived from the OSRS Wiki (MediaWiki API) plus hand-encoded 2026-meta files.
Published from the main repo by `scripts/publish-kb.sh`; the plugin hot-loads newer
`kbVersion`s from this repo's raw URL at startup.

Files: manifest.json (version), quests.json, recipes.json, sources.json,
goals.json, xp_rates.json, benchmarks.json, slayer.json, goal-node.schema.json.
